# Roadmap

## Near term

- Add subcommands like `gemini aws design` and `gemini k8s analyze`
- Add structured JSON output mode
- Add retry/backoff for rate limits
- Add unit tests around prompt assembly and error formatting

## Mid term

- Add GitHub Actions for linting and packaging
- Add richer markdown rendering in terminal
- Add file-type aware prompt templates

## Long term

- Add a small plugin system for AWS CLI, kubectl, and log parsers
- Add packaging with `pyproject.toml`
- Add an optional enterprise mode using newer Gemini interaction patterns

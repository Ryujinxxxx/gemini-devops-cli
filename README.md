## 🔥 Overview

**Gemini DevOps CLI** is a production-style AI assistant that brings **Google Gemini AI directly into your terminal**.

It enables engineers to analyze logs, debug Kubernetes, design AWS architecture, and review infrastructure — all from CLI.

---

## ⚡ Key Features

- 🧠 AI-powered CLI assistant
- ☁️ AWS architecture & troubleshooting
- ☸️ Kubernetes debugging & analysis
- 🐧 Linux system diagnostics
- 📄 File & config review (YAML, logs, Terraform)
- 🔎 Real-time knowledge with grounding
- 📊 Command output analysis (`kubectl`, `aws`, logs)
- 💬 Interactive chat mode with memory
- ⚡ Streaming responses
- 🎯 Mode-based intelligence
